📘 Book Exchange & Resource Sharing System
📌 Description

The Book Exchange and Resource Sharing System is a web-based application designed to help students and communities share, exchange, and manage books efficiently.

This project enables users to browse books, request exchanges, and manage their inventory, creating a collaborative learning ecosystem without requiring a backend server.

🌍 Community Impact

This project is built to support:

📚 Students who cannot afford books
🔄 Peer-to-peer book sharing within campuses
🌱 Promoting reuse and reducing waste

It helps create a community-driven platform for knowledge sharing.

🚀 Features
🔐 User Authentication (Login & Register)
📖 Book Listing & Management
🔍 Real-time Search & Filtering
🔄 Book Exchange Request System
📊 User Dashboard with statistics
🧑‍💼 Admin Panel for system management
🌙 Light/Dark Mode UI
💬 Community Forum (Simulated)
🤖 AI-based Book Recommendations (Simulated)
🏗️ Architecture

The system follows a modular client-side architecture:

UI Layer: HTML5, Bootstrap
Logic Layer: JavaScript (ES6 Modules)
Storage Layer: LocalStorage (simulated database)

This design ensures scalability and maintainability.

🛠️ Tech Stack
HTML5
CSS3 & Bootstrap 5
JavaScript (ES6+)
LocalStorage API
Boxicons
Google Fonts
🔄 How It Works
Users register/login
Add books to their inventory
Browse books shared by others
Send exchange requests
Owners approve/reject requests
Book exchange is completed
